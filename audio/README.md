# MIDI audio assets

- `GeneralUser-GS.sf2` — GeneralUser GS 2.0.3 by S. Christian Collins, downloaded unmodified from the [canonical repository](https://github.com/mrbumpy409/GeneralUser-GS). See `GeneralUser-GS-LICENSE.txt`.
- `MuseScore_General.sf3` — official MuseScore General release from the [OSU Open Source Lab mirror](https://ftp.osuosl.org/pub/musescore/soundfont/MuseScore_General/). See `MuseScore-General-LICENSE.md`.
- `TimGM6mb.sf2` — unmodified bank extracted from Debian package `timgm6mb-soundfont_1.3-5_all.deb`. See `TimGM6mb-COPYRIGHT.txt` and `TimGM6mb-GPL-2.0.txt`.
- `midi-audio-engine.min.js` — browser bundle built from `src/midi-audio-engine.js` with the pinned npm dependencies.
- `spessasynth_processor.min.js` — AudioWorklet processor copied from the installed `spessasynth_lib` package. See `SPESSASYNTH-LICENSE.txt`.

Timbres of Heaven is intentionally not bundled. Its archive prohibits redistribution without written permission; users can download it from MidKar and load their extracted `.sf2` copy through `🎛 Sounds`.
